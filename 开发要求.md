## 一、项目概述

飞书待办提醒 AstrBot 插件。用户通过飞书私聊向机器人发送任务详情（文本或截图），LLM 自动提取任务信息创建飞书待办，并设置多级 DDL 提醒。

## 五、main.py 开发规范

### 5.1 类结构

- 继承 `astrbot.api.star.Star`
- `__init__(self, context: Context)` — 最小化初始化，不在此处做重 IO
- `async initialize(self)` — 插件的异步初始化入口：
  - 加载配置
  - 实例化 FeishuTaskClient
  - 调用 `self.context.add_llm_tools()` 注册 Function Tool
  - 启动定时巡检任务
- `async terminate(self)` — 取消定时任务，释放资源

### 5.2 Function Tool

- **禁止**使用模块级全局变量（如 `_star = None`）传递 Star 实例
- 使用 `@dataclass` + `FunctionTool[AstrAgentContext]` 方式定义
- `name` 固定为 `create_reminder_task`
- `description` 须用中文详细描述工具用途、触发条件、参数提取规则
- `parameters` 的每个字段 `description` 须用中文，明确类型和语义
- 注册时机：`initialize()` 中调用 `self.context.add_llm_tools()`
- `call` 方法通过 `context.context.event` 获取 `AstrMessageEvent` 及其 `unified_msg_origin`

### 5.3 定时巡检

- 使用 asyncio 或 APScheduler 实现
- 扫描间隔：30 分钟
- 需 try-except 包裹，单条失败不影响其余
- 推送消息使用 `self.context.send_message(umo, chain)`
- 消息链使用 `MessageChain().message("...")` 构建

### 5.4 时区

- 解析用户输入的截止时间时，须显式处理 `Asia/Shanghai`
- 所有内部时间戳统一使用 UTC 毫秒
- 巡检比较时须确保时间基准一致

## 六、feishu_api.py 开发规范

### 六、I/O 规范

- 仅使用异步 HTTP 库：`httpx` 或 `aiohttp`（**禁止** `requests`）
- 封装为 `FeishuTaskClient` 类
- 提供方法：
  - `create_task(summary, due_timestamp_ms, assignee_open_id, description, reminder_minutes)` → `dict`
  - `list_tasks(page_size)` → `list`
- `tenant_access_token` 须缓存并提前 120 秒刷新
- API 调用须设置超时（建议 15 秒）
- 返回前须检查 `code == 0`，否则抛出明确异常

### 凭据获取

- `app_id` 和 `app_secret` 由 `main.py` 在初始化时从配置中读取后传入，不在 `feishu_api.py` 内直接读取文件或环境变量

---

## 七、配置规范

- 配置文件路径：`data/plugin_data/astrbot_plugin_feishu_todo/config.json`
- 由插件在 `initialize()` 中自动生成默认模板（空凭证），用户手动填写
- 不在插件代码目录内存放任何配置文件
- 配置内容：

```json
{
  "feishu_app_id": "",
  "feishu_app_secret": ""
}
```

---

## 八、数据持久化

- 使用 AstrBot 内置 KV 存储（`self.put_kv_data` / `self.get_kv_data`）
- 键名规范：`task_{guid}` 存储单任务规则，`_task_ids` 存储所有活跃任务 GUID 列表
- 结构化数据须 JSON 序列化后再存储

---

## 九、代码风格

- Python 类型注解覆盖所有函数签名
- 使用 `ruff` 格式化代码（提交前执行 `ruff check --fix . && ruff format .`）
- 日志使用 `astrbot.api.logger`，重要操作（创建任务、推送提醒、异常）须打印日志
- 注释使用中文
- 异步全链路，禁止在 async 函数内调用同步阻塞方法
